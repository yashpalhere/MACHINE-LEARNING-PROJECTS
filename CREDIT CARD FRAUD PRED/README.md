# Credit Card Fraud Detection

A machine learning project focused on detecting fraudulent credit card transactions using classification models and imbalance handling techniques.

## What I Did
- Explored and analyzed the dataset
- Observed severe class imbalance
- Trained a baseline classification model
- Evaluated model performance using multiple metrics
- Applied SMOTE to balance the dataset
- Compared performance before and after balancing

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- imbalanced-learn

## Techniques Used
- SMOTE
- Train-test split
- Classification metrics evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Learnings
- Imbalanced dataset handling
- Precision vs Recall tradeoff
- Importance of Recall in fraud detection
- Why accuracy can be misleading
- Business-oriented ML evaluation
