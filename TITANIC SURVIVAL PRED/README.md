# Titanic Survival Prediction

A machine learning project based on the Kaggle Titanic dataset to predict passenger survival.

## What I Did
- Cleaned and preprocessed the dataset
- Handled missing values
- Removed unnecessary columns
- Applied categorical encoding using get_dummies
- Split data into training and testing sets
- Trained a Decision Tree Classifier

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Model Used
- DecisionTreeClassifier
