# House Prices Prediction

A machine learning project based on the Kaggle House Prices dataset.

## What I Did
- Cleaned and preprocessed the data
- Handled missing values
- Worked with categorical features
- Trained a Random Forest Regressor
- Tuned parameters using RandomizedSearchCV

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
