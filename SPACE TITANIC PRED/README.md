# Spaceship Titanic Prediction

A machine learning project based on the Kaggle Spaceship Titanic dataset to predict whether passengers were transported to another dimension.

## What I Did
- Cleaned and preprocessed the dataset
- Handled missing values
- Worked with categorical and numerical features
- Applied encoding techniques
- Performed feature engineering
- Trained multiple classification models
- Tuned model parameters using RandomizedSearchCV

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- XGBoost

## Models Used
- Logistic Regression
- Random Forest Classifier
- XGBoost Classifier
